Key workers: apply for a coronavirus test

If you're a key worker who is self-isolating, you can now take a coronavirus test to see if you can return to work.

If your test result is negative, it means:

  • you don't currently have coronavirus
  • you can safely return to work, providing anyone you live with also tests negative

Before you start

To complete this form you will need:

  • registration number of vehicle you plan to attend the test
  • your National Insurance Number

Follow these next steps to check you are eligible for the coronavirus test. We will then ask for a few details about yourself, before you can got on to register for the test.
