Test for Coronavirus
(SARS-CoV-2)

The new coronavirus is called SARS-CoV-2. In some people it leads to the disease COVID-19.

What is the test for?

If your test result is negative, it means:

  • you don't currently have coronavirus
  • you can safely return to work, providing anyone you live with also tests negative

If your test positive, or someone you live with tests positive, you cannot return to work. You'll need to continue to self-isolate. Read NHS advice about how long to self isolate.

Key workers can now be tested

If you're a key worker who is self-isolating (not able to leave home), you can apply for you and your household to get tested for coronavirus.

Follow these next steps to check you are eligible for the coronavirus test. We will then ask for a few details about yourself, before you can go on to register for the test.

If you're not a key worker

If you're not a key worker you cannot apply for a test. If you have symptoms, use the 111 online coronavirus service to find out what to do.
