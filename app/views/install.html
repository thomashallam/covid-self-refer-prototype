Using this kit

Get a basic set up first

  • Read the gov.uk installation guide - https://govuk-prototype-kit.herokuapp.com/docs
  • Read nhs.uk installation guide - https://nhsuk-prototype-kit.azurewebsites.net/docs/install
  • Install heroku cli: https://devcenter.heroku.com/articles/heroku-cli
  • Install github: https://git-scm.com/downloads
  • Test installing and running a new version of the gov.uk kit
  • Using git in Cli: https://nhsuk-prototype-kit.azurewebsites.net/docs/how-tos/git
  • Publishing to heroku

Working on this kit

  • Clone/Download/Fork a copy of this version

Notes:

If you typically use the nhs.uk kit...

The gov.uk kit needs different instructions in Terminal. Type npm start to start the kit.

The format for setting an Heroku password is different for the nhs.uk kit to gov.uk kit

The nhsuk and govuk kits have different markup

Govuk Service Manual / Pattern library https://design-system.service.gov.uk/components/

NHSuk Service Manual / Pattern library https://service-manual.nhs.uk/design-system/components

NHSuk Service examples - https://github.com/nhsuk/nhs-prototypes-list

Other design systems across gov: https://design.homeoffice.gov.uk/
