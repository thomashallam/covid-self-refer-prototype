    1.6 - eligibility focus
  • ensure the whole service is easy to understand for all users
  • ensure users don't confuse who is eligible to apply for a test
  • ensure users don't confuse whether or not they need to add household members
  • ensure there are sufficient clinical safety processes are in place

    • 1.7 - home testing / assisted digital focus
  • allow key workers without a car to get tested conveniently
  • allow key workers without internet access to get tested
  • allow key worker who need home testing to order a test kit
  • allow key worker who need home testing to register a test kit
  • allow key workers with queries about home testing to resolve these

    • 1.8 - appointment uplift
  • provide flexibility to key workers if their scheduling changes
  • support same day testing - very high priority (i.e. no slot booking needed)
